========================================================================
    BIBLIOTHÈQUE MICROSOFT FOUNDATION CLASS : Vue d'ensemble du projet 
    RepItemsWrap
========================================================================


AppWizard a créé cette DLL RepItemsWrap à votre intention.  
Cette DLL décrit les principes de base de l'utilisation de 
Microsoft Foundation Classes et vous permet de créer votre DLL.

Ce fichier contient un résumé du contenu de chacun des fichiers qui 
constituent votre DLL RepItemsWrap.

RepItemsWrap.vcproj
    Il s'agit du fichier projet principal pour les projets VC++ générés 
    à l'aide d'un Assistant Application. 
    Il contient des informations sur la version de Visual C++ utilisée 
    pour générer le fichier, ainsi que des informations relatives aux 
    plates-formes, configurations et fonctionnalités du projet que vous 
    avez sélectionnées dans l'Assistant Application.

RepItemsWrap.h
    Il s'agit du fichier d'en-tête principal pour la DLL.  Il déclare la 
    classe CRepItemsWrapApp.

RepItemsWrap.cpp
    Il s'agit du fichier source principal de la DLL.  Il contient la 
    classe CRepItemsWrapApp.

RepItemsWrap.rc
    Il s'agit de la liste de toutes les ressources Microsoft Windows que
    le programme utilise.  Elle comprend les icônes, les bitmaps et les
    curseurs qui sont stockés dans le sous-répertoire RES.  Ce fichier 
    peut être modifié directement dans Microsoft Visual C++.

res\RepItemsWrap.rc2
    Ce fichier contient les ressources qui ne sont pas modifiées par 
    Microsoft Visual C++.  Vous devez placer toutes les ressources non 
    modifiables par l'éditeur de ressources dans ce fichier.

RepItemsWrap.def
    Ce fichier contient des informations sur la DLL qui doivent être
    fournies pour qu'elle s'exécute avec Microsoft Windows.  Il définit 
    des paramètres tels que le nom et la description de la DLL.  Il 
    exporte également des fonctions de la DLL.

/////////////////////////////////////////////////////////////////////////////
Autres fichiers standard :

StdAfx.h, StdAfx.cpp
    Ces fichiers sont utilisés pour générer un fichier d'en-tête précompilé 
    (PCH) nommé RepItemsWrap.pch et un fichier de type précompilé 
    nommé StdAfx.obj.

Resource.h
    Il s'agit du ficher d'en-tête standard, qui définit les nouveaux ID de 
    ressources.
    Microsoft Visual C++ lit et met à jour ce fichier.
