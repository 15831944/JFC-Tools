Ajout d'une classe: CTarifAnnée:
--------------------------------
Cette classe contient les tarifs de l'année de chacun des produits,bacs,complexes définit dans un fichier
Chaque fichier de tarif est chargé sous la forme d'une instance de cette classe
  
import du tarif des complexes,bacs et produits.
On réalise une fonction d'import(ImportFile()) qui sera appelée par plusieurs menu, elle prend en paramètre un nom de fichier d'import et vérifie le type de ce fichier
avant de le charger dans l'application.
import des saisonnalités + nbentree par complexe.

Modification de CFichierCinema: 
--------------------------------------
ajout d'une variable de type CString(m_separator) indiquant le séparateur de fichier.
ajout d'une variable de type CString(m_delimitor) indiquant le délimiteur de fin de fichier.

Modification de CBaseCinema: 
--------------------------------------
ajout d'un fichier LockData définissant l'autorisation d'accéder à la base.
Ce fichier est ouvert en mode create dans le savetable en excluant tout autre accès au fichier.
Ce même fichier est ouvert en excluant l'écriture dans le loadtable.

Modification de CSaisonnalité: 
------------------------------------
définition d'un copie constructor et = operator
ajout d'une variable nom de fichier à la sasonnalité
ajout d'une méthode SetSaisonnalite(int saisonnalite,int pos) permettant de modifier la saisonnalite

Modification des tables:
------------------------

pour répérer leur modification et ne sauvegarder que les tables modifiées.

gestion de clés dans la base de registre:OK
