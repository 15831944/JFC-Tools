CObj_LBox

La classe CObj_LBox dérive de la classe CListBox. Elle fonctionne comme un control listbox, avec la possibilité d'une deuxième sélection, ou d'une sursélection.

Implantation

Il faut inclure dans le projet :

  • le fichier d'en-tête OBJ_LBOX.H
  • le fichier source OBJ_LBOX.CPP

Il faut inclure dans le fichier d'en-tête appellant l'objet :

  • #include <OBJ_LBOX.H>
  • Modifier le type ListBox du control en CObj_Lbox

 

!!!!!    SI ON UTILISE LE MESSAGE OnSelChange il faut obligatoirement dans une lb multi sélection traiter OnDblClk !!!

CObj_LBox::Initialise
void Initialise(short Style, short ModeSel1=ETENDUE_JFC, short ModeSel2=ETENDUE_JFC, COLORREF ColorSel1=0, COLORREF ColorSel2=0, short SelectRules=RIGHTINTOLEFT, COLORREF BkColor=0, CFont *TxtFont=NULL, COLORREF ColorTxt=0);

Paramètres

Style    Spécifie l'un des styles suivant :

  • UNISTATE : sélection d'un item par click gauche seulement.
  • DBLSTATE : sélection d'un item par click gauche ou droit
  • TRISTATE : sélection d'un item par click gauche et/ou droit

ModeSel1 et ModeSel2    Spécifie l'un des formats suivant; -1 si aucun.

  • SIMPLE : Sélection d'un champ
  • MULTIPLE : Sélection de plusieurs champs
  • ETENDUE : Sélection d'une vague
  • ETENDUE_JFC : Sélection de plusieurs vagues

ColorSel1    Couleur RGB de la sélection normal (BLEU par défaut); Nulle si non utilisée.

ColorSel2    Couleur RGB de la seconde sélection (ROUGE par défaut); Nulle si non utilisée.

SelectRules    Règles des sélections :

  • INDEPENDANTE : les deux sélections sont indépendantes. Elles ne peuvent pas se superposées.
  • CROISEE : les deux sélections sont libres. Elles peuvent se superposées.
  • LEFTINTORIGHT : la sélection click gauche n'est possible qu'a l'intérieur de la sélection click droite.
  • RIGHTINTOLEFT : la sélection click droite n'est possible qu'a l'intérieur de la sélection click gauche.
  • EXCLUSIVE : L'une, ou l'autre, mais pas les deux.

BkColor    Couleur RGB du fond (BLANC par défaut).

TxtFont    Font du texte.

ColorTxt    Couleur RGB du texte.

Remarque

Initialise les paramètres généraux de l'objet.

CObj_LBox::OnItemState
virtual void OnItemState(short nItem, shortState);

Paramètres

nItem    Numéro de l'item.

State    Etat de l'item :

  • 0 : Item non sélectionné.
  • 1 : Item sélectionné click Gauche
  • 2 : Item sélectionné click Droit
  • 4 : Item sursélectionné

Remarque

Appelé lorsqu'un item change d'état.

CObj_LBox::GetSelCount
int GetSelCount();

Valeur de retour

Le nombre d'éléments sélectionnés; 0 si aucun.

CObj_LBox::GetSelItems
int GetSelItems(int nMaxItems, LPINT rgIndex );

Valeur de retour

Le nombre d'éléments sélectionnés qui correspond à la taille du buffer rgIndex.

Paramètres

nMaxItems Nombre maximal d'items sélectionné que peut stockée le buffer rgIndex.

rgIndex Buffer pré-alloué où seront récupéré les indices des éléments sélectionnés.

Remarque

Récupère la liste des élements sélectionnés dans un buffer pré-alloué.

CObj_LBox::ChangeMode
void ChangeMode(short ModeSel1, short ModeSel2, bool flagRedraw=true);

Paramètres

ModeSel1 Nouveau mode de sélection correspondant à la sélection 1; Voir Initialisation(...).

ModeSel2 Nouveau mode de sélection correspondant à la sélection 2; Voir Initialisation(...).

flagRedraw Flag décidant de la prise en compte immédiate(true) des nouveaux mode, ou de leur ignorance(false).

Remarque

Changement du mode de sélection 1 et 2, avec rafrachissement ou non.

CObj_LBox::ChangeSelectRules
void ChangeSelectRules(short newSelectRules);

Paramètes

newSelectRules Nouvelle règle de sélection de l'objet. Voir Initialise(...) pour les différentes valeurs.

Remarque

Change la règle de sélection des élements de l'objet.

CObj_LBox::ChangeColorSel1
void ChangeColorSel1(COLORREF newcolor, bool flagRedraw=true);
CObj_LBox::ChangeColorSel2
void ChangeColorSel2(COLORREF newcolor, bool flagRedraw=true);

Paramètres

newcolor Nouvelle couleur adapté à la sélection 1 ou 2.

flagRedraw Flag décidant de la prise en compte immédiate(true) des nouvelles couleurs de sélections, ou de leur ignorance(false).

Remarque

Modifie la couleur de la sélection 1 ou 2.
