CObj_Tab

La classe CObj_Tab dérive de la classe CStatic.

Implantation

Il faut inclure dans le projet :

  • le fichier d'en-tête OBJ_TAB.H
  • le fichier source OBJ_TAB.CPP
Il faut inclure dans le fichier d'en-tête appellant l'objet :
  • #include <OBJ_TAB.H>
  • Modifier le type CStatic du control en CObj_Tab

Membres de la class CObj_Tab

Initialisation
 
Initialise Initialise les paramètres généraux de l'objet.
InitialiseTitre Initialise chacun des modules du tableau séparément.
SetTitre Passe les éléments de chacun des titres séparément à l'objet 
SetTableau Passe les éléments du tableau à l'objet.
SetFormatTitre Configure le format d'une cellule dans un titre.
SetFormatCellule Configure le format d'une cellule dans le tableau.
PutMsgInToTab Affichage d'un message dans le fond du tableau.

Position
 
SetPosition Met à jour l'affichage suivant la nouvelle position passé en  colonne/ligne.

Sélection
 
SelectTitre Sélection d'un élément d'un titre. Utilisé pour les tris.
SetSelection Sélection d'un élément d'un des modules.
RAZSelection Désactive toutes les sélections de chaque module.

Ascenseur
 
MAJPosScrollV Mise à jour de la position du calendrier suivant la position de l'ascenseur verticale.
MAJPosScrollH Mise à jour de la position du calendrier suivant la position de l'ascenseur horizontale.

Libération
 
LibereObjet Libération des allocations mémoire.

Surdéfinition de fonction
 
OnSort Signal la sélection d'un élément d'un titre pour effectuer un tri.

CObj_Tab::Initialise
void Initialise(short Style, short Format, short DataType, shortdxCel, short dyCel, short nLigVisu, shortnColVisu, CScrollBar * ScrollH, CScrollBar * ScrollV, CFont * pFont=NULL, COLORREF TabColorBg=0);

Paramètres

Style    Spécifie l'un des styles suivant :

  • STYLE_TAB_DEFAUT : Visualisation de données
  • STYLE_TAB_SELECTION : Sélection de colonnes/titres/cellules
  • STYLE_TAB_EDIT : Edition des cellules. <<<<<<< PAS ENCORE DISPONIBLE >>>>>>>
        Des options peuvent être jointe au style :
  • OPT_TAB_NOGRIL_COL : Pas de séparation entre les colonnes
  • OPT_TAB_NOGRIL_LIG : Pas de séparation entre les lignes
  • OPT_TAB_SELECT_COL : Sélection par colonne
  • OPT_TAB_SELECT_LIG : Sélection par ligne
  • OPT_TAB_SELECT_CEL : Sélection par cellule
  • OPT_TAB_TRI_COL : Tri par colonne
  • OPT_TAB_TRI_LIG : Tri par ligne
  • OPT_TAB_NOGRIL_OUT : Pas de séparateurs dans les espaces libres de l'objet.
Format    Spécifie l'un des formats suivant:
  • FORMAT_TAB_BTN
DataType    Spécifie le type des données qui seront afichées.

dxCel    Unités en pixel de la largeur par défaut des cellules.

dyCel    Unités en pixel de la hauteur par défaut des cellules.

nLigVisu    Nombre de ligne contenue dans l'espace affiché.

nColVisu    Nombre de colonne contenue dans l'espace affiché.

ScrollH    Pointeur de l'ascenseur horizontal lié à l'objet.

ScrollV    Pointeur de l'ascenseur vertical lié à l'objet.

pFont    Font utilisé pour les éléments de l'objet.

TabColorBg    Couleur du fond.

Remarque

Cette fonction configure les paramètres globaux de l'objet. Aucune configuration par défaut n'est disponible.

CObj_Tab::InitialiseTitre
void InitialiseTitre(short TypeTitre, short dxTitre, short dyTitre, CFont * pFont=NULL, COLORREF TitreColorBg=0);

Paramètres

TypeTitre    Spécifie l'un des modules suivant  :

  • TITRE_LIG : Ce module correspond à la partie gauche de l'objet.
  • TITRE_COL : Ce module correspond à la partie haute de l'objet.
  • CORPS : Ce module correspond à la partie centrale de l'objet.
        Dans le futur, cet objet sera ammené à posséder d'autres modules, du genre multi-titre, titre-total, des sous-titres, etc...

dxTitre    Nombre de pixels minimum de la largeur des cellules du module. La largeur est automatiquement retaillé si un éléments est plus large.

dyTitre    Nombre de pixels minimum de la hauteur des cellules du module.

pFont    Font utilisé pour les éléments du module.

TitreColorBg    Couleur de fond du module.

Remarque

Fonction permettant d'initialiser les paramètres pour chacun des modules.

CObj_Tab::SetTitre
void SetTitre(short TypeTitre, CStringArray * TabTitre);

Paramètres

TypeTitre    Spécifie l'un des titres suivant :

  • TITRE_LIG : Ce module correspond à la partie gauche du tableau.
  • TITRE_COL : Ce module correspond à la partie haute du tableau.
TabTitre    Pointeur représentant le tableau des éléments du titre spécifié.

Remarque

Défini les éléments qui vont apparaître dans les barre de titres.

void SetTableau(short NbLig, CStringArray * TabElements);

CObj_Tab::SetFormatTitre
void SetFormatTitre(short TypeTitre, short NroTitre, short dxTitre, short dyTitre, UINTOrientation, CFont * pFont=NULL, COLORREF ColorCel=0, bool flag=false);

Paramètres

TypeTitre    Voir CObj_Tab::SetTitre

NroTitre    Spécifie l'élément du titre. De 0 à n. Ou -1 pour désigner la barre de titre et tous ces élements.

dxTitre    Spécifie la largeur en pixel de la cellule ou apparait l'élément.

dyTitre    Spécifie la hauteur en pixel de la cellule ou apparait l'élément.

Orientation    Spécifie la position de l'élément dans sa cellule. Voir le paramètre nFormat de DrawText.

pFont    Font utilisé pour affiché cet élément.

ColorCel    Couleur utilisé pour peindre la cellule de cet élément.

flag    Met à jour l'affichage si true.

Remarque

Configure le format d'un élément et sa cellule, pour un titre spécifié.

Paramètres

CObj_Tab::LibereObjet
void LibereObjet(void);

Remarque

Libération mémoire des variables qui contiennent les éléments des différents modules de l'objet, et autres allocations.
Cette fonction doit être appelé avant que l'objet disparaisse.
