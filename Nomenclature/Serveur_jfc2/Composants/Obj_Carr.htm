Implantation

Il faut inclure dans le projet :

  • le fichier d'en-tête OBJ_CARR.H
  • le fichier source OBJ_CARR.CPP
  • #include <OBJ_CARR.RCO> dans la ressource principale

Il faut inclure dans le fichier d'en-tête appellant l'objet :

  • #include <OBJ_CARR.H>
  • Une variable de type CObj_carr

Appel :

void Initialise(short Style, CByteArray * Selections);

short DonneResult(CByteArray * Selections);

Dans les deux fonctions ci-dessus, "Selections" est un tableau d'octets représentant la sélection pour chaque ligne.
Le bit 0 représente le lundi, le bit 6 représente le dimanche.

Styles

CARRS_STANDARD  style par defaut
CARRS_DEMINEUR  style démineur
