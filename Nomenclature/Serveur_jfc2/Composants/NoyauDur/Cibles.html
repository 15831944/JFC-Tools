CCibleUnivers    

La classe CCibleUnivers est utilisée pour charger différentes données référencées dans les fichiers cibles.

Implantation

Il faut inclure dans le projet :

  • La classe CCibleUnivers : CibleUnivers.cpp et CibleUnivers.h
  • La classe CTableCIBLE : TableCible.cpp et TableCible.h
  • Le composant CFileUtils : FileUtils.cpp et FileUtils.h
Il faut inclure dans le fichier d'en-tête appelant l'objet #include "cibleunivers.h". 

Membres de la class CCibleUnivers

CCibleUnivers Construit un objet cible.

Initialisation Initialisation d'un objet cible.

CCibleUnivers::CCibleUnivers
CCibleUnivers(void);
CCibleUnivers(CString EtudIni, CString Marche);

Paramètres
EtudIniChemin du fichier Ataaetud.ini
MarcheMarché des cibles

Remarque
Le premier constructeur déclare un objet CCibleUnivers.
Le second déclare et initialise les paramètres pour les cibles.

ATTENTION!!!
La libération de l'objet CCibleUnivers entrainera l'invalidité du pointeur retourné.

CCibleUnivers::Initialisation
bool Initialisation(CString EtudIni, CString Marche);

Valeur de retour
TRUE si le fichier d'initialisation est valide; FALSE sinon.

Paramètres
EtudIniChemin du fichier Ataaetud.ini
MarcheMarché à charger

Remarque
Initialise le marché du noyau dur, vérifie l'existence du fichier INI.
Retourne FALSE si EtudIni est invalide.

CCibleUnivers::DonneTableCibles
CTableCibleArray &DonneTableCibles(int Univers);

Valeur de retour
L'adresse d'un tableau de type CArray contenant la liste des cibles.

Paramètres
Univers Le numéro de l'univers à charger.
Remarque
Le pointeur retourné est valable TANT QUE l'objet CCibleUnivers déclaré existe.
